FAKE TODO API

Fake todos for your personal projects

@temalcode GitHub

About

Fake todos for your personal projects. Created using NodeJS and Express JS. Hosted on Heroku. Supports only for GET requests. No "CORS" (Cross-origin resource sharing) error.

Getting Started

Get all todos:

http://localhost:5000/api/

Get a todo by id:

http://localhost:5000/api/:id

Get a random todo:

http://localhost:5000/api/random

Get completed todos:

http://localhost:5000/api/completed

Get not completed todos:

http://localhost:5000/api/notcompleted
